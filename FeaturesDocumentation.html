def LeerLC_MACHO(fid):
    saltos_linea = 3
    delimiter = ' '
    for i in range(0,saltos_linea):
        fid.next()
    LC = []

    for lines in fid:
        str_line = lines.strip().split()
        floats = map(float, str_line)
        #numbers = (number for number in str_line.split())
        LC.append(floats)

    LC = np.asarray(LC)
    return LC

Standard deviaton \(\sigma\)

The \(\sigma\) is the standard deviation defines as

\[ \sigma = \frac{(y-\hat{y})^2}{N-1} \]
